# oboni — AI Model Portfolio

> A personal portfolio site for **oboni**, an AI model specialising in news reading, storytelling, and digital media presentation.

## 🌐 Live Pages

| Page | Description |
|------|-------------|
| [`index.html`](index.html) | Hero / home page with profile and quick info |
| [`skills.html`](skills.html) | Skills & proficiency levels |
| [`projects.html`](projects.html) | Broadcast appearances & featured work |
| [`contact.html`](contact.html) | Contact & social links |

## ✨ Features

- Dark, modern UI with a cyan accent colour scheme
- Animated hero section with profile image
- Skill bars with scroll-triggered animation
- Responsive layout (mobile & desktop)
- Smooth fade-up entrance animations
- Interactive project cards with hover effects
- Typing-effect role indicator on the homepage

## 🚀 Getting Started

No build step required — this is a fully static site.

1. Clone the repository:
   ```bash
   git clone https://github.com/swapnilchowdhury-2003/ai-model.git
   ```
2. Open `index.html` in any modern browser, or serve with any static file server:
   ```bash
   npx serve .
   ```

## 🎨 Tech Stack

- **HTML5** — semantic markup
- **CSS3** — custom properties, grid, flexbox, keyframe animations
- **Vanilla JS** — Intersection Observer for scroll animations, typing effect
- **Google Fonts** — Syne (headings) & Hind Siliguri (body)
